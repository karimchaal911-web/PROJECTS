# Projects Portfolio

Welcome to my projects portfolio.

This repository contains the projects I have completed throughout my academic journey, as well as future projects I will continue to build as I improve my technical skills.

The main goal of this repository is to document my progress, organize my work, and showcase practical applications of the concepts I study in engineering, data analysis, programming, and artificial intelligence.

## Main Areas

This portfolio mainly includes projects related to:

- Data Analysis
- Data Visualization
- Machine Learning
- Artificial Intelligence
- Arduino and embedded systems
- Python programming
- Academic engineering projects

## Repository Purpose

This repository is designed to be a central place where I collect and organize my work. Each project will include, when possible:

- A clear project description
- The problem or objective
- The tools and technologies used
- The dataset or hardware used
- The methodology followed
- The results obtained
- Notes, limitations, and possible improvements
